Pediatric Center Website is a modern, responsive web application built to showcase pediatric healthcare services in an engaging and user-friendly way. The project is developed using React.js, integrated with Fetch API for dynamic content loading, and enhanced with smooth sliding animations to deliver a polished and interactive user experience.
This application demonstrates clean component architecture, efficient state management, API-driven rendering, and visually appealing motion effects suitable for real-world healthcare platforms.
